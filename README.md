# 🌱 Luke's Garden Planner 2026 — Decatur, GA (Zone 8a)

An interactive, single-file HTML garden planner for the 2026 growing season. Tracks three raised beds, grow bags, hydroponic systems (AeroGarden, Gardyn, 9-port unit), a greenhouse, and perennials — with tabbed views for overview, beds, hydro, timing, varieties, and spring history.

**Open `index.html` in any modern browser. No server required.**

| Flat parsley | ⅛–¼" |
| Dill, thyme, sage, oregano | Surface sow (light-dependent) |

## Key Principles

- **Okra:** direct sow only — dislikes root disturbance
- **Peony:** divide in fall, never spring; use 3–5 eye sections, plant 1–2" deep
- **Liriope species matter:** *L. spicata* spreads via stolons; *L. muscari* stays clumping
- **Citrus:** grafted 2–3 year old trees fruit fastest (Four Winds Growers recommended)
- **Basil:** distributed as companion planting rather than dedicated bed space
- **Snake plants:** not frost-hardy in Zone 8a — indoor or covered porch only
